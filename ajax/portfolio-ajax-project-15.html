Sweettooth

Project Info:

This is a mini online store for fast food, icecreams, candies,...

This is the fully functional and interactive frontend version of the app. It is very responsive across all devices. The data in display is being fetched from the Timbu API which is part of a project in one of my internships.

Project Details:
  • Technologies:ReactJS, Typescript, Vite, SASS, Context API, Timbu Cloud API
