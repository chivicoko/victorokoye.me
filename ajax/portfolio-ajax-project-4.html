Flavor Point - Recipe App

Project Info:

This is a recipe app where one can easily find the recipe to different meals, including meals of different cultures and traditions like American, Iitalian, Thai, Japanese... The meals are been fetched from the wonderful spoonacular api that I found online. I added an all round search functionality to this recipe app for easy search throughout the app.
